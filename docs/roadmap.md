# Birthday Bot v3 — Development Roadmap

This roadmap is indicative and may change.
There is no fixed timeline.

---

## Phase 0 — Groundwork

- repository setup
- documentation
- architecture validation
- issue backlog definition

---

## Phase 1 — Core Foundations

- database schema
- celebration model
- scheduler per guild
- soft deletion support

Goal: stable core with no Discord output yet.

---

## Phase 2 — Message Rendering & Delivery

- template engine
- variable resolution
- channel & role resolution
- webhook delivery with fallback

Goal: first end-to-end celebration execution.

---

## Phase 3 — Discord Commands

- /birthday commands
- /events commands
- /premium commands
- minimal server config

Goal: usable bot without dashboard.

---

## Phase 4 — Premium & Integrations

- Patreon webhook handling
- quota enforcement
- restricted mode
- DM notifications
- top.gg vote rewards (temporary roles)

---

## Phase 5 — Dashboard

- internal API setup
- Discord OAuth authentication
- dashboard configuration flows
- premium visibility & restrictions

---

## Phase 6 — Stabilization

- logging & monitoring
- memory profiling
- migration tooling (if applicable)

---

## Notes

- phases may overlap
- features may be delayed or dropped
- stability is always the priority
